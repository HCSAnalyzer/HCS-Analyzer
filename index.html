    HCS Analyzer

    Project maintained by HCSAnalyzer Hosted on GitHub Pages — Theme by mattgraham

    Welcome to HCS Analyzer GitHub Pages.

    HCS Analyzer is an open source application dedicated to High Content Screening (HCS) mutlivariate data processing and analysis.

    This is a Microsoft Windows application originally developed in C#. The main goal of this application is to provide the screening community a user-friendly application where one can develop and share easily its own project via a plugins manager.

    System Requirements: MS windows .NET 4.0 framework is required.
